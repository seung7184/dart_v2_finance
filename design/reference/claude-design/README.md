# Claude Design Reference Files

These files are preserved from the Claude Design export as visual/design reference for Dart Finance brand and shell work.

They are not production app code. Do not import these files directly into the web or mobile app.

Use them only to inspect visual direction, logo geometry, token ideas, and layout concepts. When a piece is needed in the app, convert it into proper TypeScript React components and app-owned CSS using the existing Dart Finance design system.

Included files:

- `dart-logo.html` - reference logo exploration and icon geometry
- `shared.jsx` - reference shared demo components
- `web-shell.jsx` - reference sidebar/shell demo composition
- `tokens.css` - reference design tokens from the export
